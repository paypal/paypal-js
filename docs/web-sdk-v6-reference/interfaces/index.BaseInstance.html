        Returns Promise<EligiblePaymentMethodsOutput>

        A promise that resolves to payment methods eligibility information

    Remarks

    This method verifies buyer and merchant eligibility by interacting with PayPal's public API to determine whether payment methods (such as PayPal or Venmo) can be used. Use this to conditionally render the appropriate payment buttons on your site.

    Example 

    const paymentMethods = await sdkInstance.findEligibleMethods();
    const isPayPalEligible = paymentMethods.isEligible("paypal");
    if (isPayPalEligible) {
      // Render PayPal button
    }
